About this website

This is my personal website, where I used to experiment a lot with web technologies.
I still do, but nowadays most of my stuff is on my GitHub page. I also teamed up with my friend Fju. We make things as KordonBleu.

This website has been self-hosted on an ARM board running Debian for a really long time (january 2014 - 17th October 2016).
Now I use a professional hosting solution (still on Debian) because my home's power would go out from time to time, which would crash the server.

It was made with (besides ❤):

  • Node.js proxied behind Nginx
  • Express & EJS
  • MongoDB & Mongoose

It is still made with ❤, but I have migrated it from Express to Koa. And improved the code. A lot.
